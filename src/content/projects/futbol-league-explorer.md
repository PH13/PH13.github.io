An interactive Shiny application that allows users to explore and analyze futbol league data through dynamic filtering, visualization, and statistical analysis. The application provides an intuitive interface for discovering patterns and insights in sports data.

## Features

- Interactive data filtering and selection
- Dynamic visualizations of player and team statistics
- Statistical summaries and comparisons
- User-friendly interface for non-technical users

This project demonstrates the value of interactive dashboards for exploratory data analysis and making complex datasets accessible to a broader audience.
